### Prerequisites

- Node.js (v14 or higher)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/boy-who-cried-wolf/niosmo-web.git
cd niosmo-web
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```
